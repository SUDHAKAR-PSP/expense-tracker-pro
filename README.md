💰 Expense Tracker Pro

A full-stack expense tracking application with analytics, authentication, and modern UI.


🚀 Features

* 🔐 Login & Register (JWT Authentication)
* 📊 Analytics Dashboard (Pie & Line Charts)
* 💰 Income & Expense Tracking
* 🔍 Search & Filter Transactions
* 💾 Local Storage Persistence
* 🎨 Premium Glass UI + Animations


🧠 Tech Stack

* React.js
* Bootstrap + CSS
* Chart.js
* Node.js + Express
* MySQL

 
⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker-pro.git
cd expense-tracker-pro
npm install
npm run dev
```
